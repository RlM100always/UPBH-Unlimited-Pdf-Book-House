# 🌟 Unlimited PDF Book House 🌟  

Unlimited PDF Book House is a versatile **eBook Library Application**! 📚  
With thousands of books available across multiple categories, it provides an engaging and user-friendly experience for book lovers.  

---

## 🚀 Features  

### 📱 User Features  
- 🔍 **Search Books**: Quickly find books by title, author, or category.  
- 📚 **Explore Categories**: Discover books organized into Science, Arts, Commerce, Engineering, Literature, and more.  
- 📝 **Request Books**: Submit requests for books directly through the app.  
- ⚡ **Offline Mode**: Download and access books without an internet connection.  
- 🌗 **Light and Dark Mode**: Switch between light and dark themes as per your preference.  
- 🔖 **Bookmark Favorites**: Save books for quick access.  

### 🔑 Admin Features  
- 📂 **Manage Books**: Add, update, and organize books in the library.  
- 🔔 **Send Notifications**: Notify users about updates or new books.  
- 📊 **Track Requests**: Manage user-submitted book requests efficiently.  

---

## 🛠 Technology Stack  

- **Language**: Java 💙  
- **UI Framework**: XML 🎨  
- **Backend**: Firebase,SQLITE , SharedPreferences 🔥  
- **Network Requests**: Retrofit & Glide 🌐
